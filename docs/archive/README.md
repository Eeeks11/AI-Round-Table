# Archive

This directory contains historical documentation, development reports, and change summaries that were created during the development and maintenance of the Multi-Model AI Deliberation System.

## Purpose

These files are archived for reference but are not needed for normal use of the system. They document:
- Migration and refactoring processes
- Bug fixes and rate limiting improvements
- Development decisions and research
- Historical project summaries

## Contents

### Configuration Migration
- **START_HERE.md** - Introduction to the model configuration simplification
- **MIGRATION_SUMMARY.md** - Details of the models.json migration
- **CHANGES_COMPLETE.md** - Complete record of configuration changes

### Rate Limiting Improvements
- **CHANGES.md** - Rate limiting and bug fixes summary
- **FINAL_RATE_LIMIT_FIX.md** - Final rate limiting implementation
- **GEMINI_RATE_LIMITS_RESEARCH.md** - Research on Gemini API limits
- **GLOBAL_LIMIT_FIX.md** - Global rate limit handling
- **PAYG_LIMITS_UPDATE.md** - Pay-as-you-go limits documentation
- **FIX_SUMMARY.md** - Summary of various fixes
- **QUICK_FIX_REFERENCE.md** - Quick reference for fixes
- **SYSTEM_UPDATE_SUMMARY.md** - System-wide update documentation

### Project Summaries
- **PROJECT_SUMMARY.md** - Comprehensive project overview from development

## When to Review

These files may be useful if you:
- Want to understand the history of the project
- Are investigating why certain design decisions were made
- Need details about past bugs and their solutions
- Are contributing to the project and want context

## Current Documentation

For current, user-facing documentation, see the main `docs/` directory and `README.md` in the project root.
